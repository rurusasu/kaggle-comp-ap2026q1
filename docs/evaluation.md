# AP_2026Q1 - Evaluation

## Evaluation Metric

Las submissions se evaluan por **R2** (R-squared / coefficient of determination).

El ganador se evalua por su performance en el dataset PRIVATE. Uds no tienen acceso a ese resultado hasta que la competencia finaliza. Se parece al resultado PUBLIC? Por que?

## Submission File

For each ID in the test set, you must predict a probability for the TARGET variable. The file should contain a header and have the following format:

```
ID,track_popularity
26266,0
26267,0
26268,0
etc...
```
