# AP_2026Q1 - Submission Format

## Format

For each ID in the test set, you must predict a value for the `track_popularity` variable.

## File Structure

The file should contain a header and have the following format:

```
ID,track_popularity
26266,0
26267,0
26268,0
etc...
```

## Evaluation

Submissions are evaluated by R2 (R-squared / coefficient of determination). The winner is determined by performance on the PRIVATE dataset, which is not visible until the competition ends.
